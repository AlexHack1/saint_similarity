    Saint Similarity Network

    This visualization displays saints as nodes in a constellation graph, positioned based on their similarity from Wikipedia content.

    How to use:

    • Search for a saint using the search box
    • Click on any saint to see similar saints
    • Scroll to zoom, drag the background to pan

    Saints closer together have more similar Wikipedia articles.

    Text embeddings are created with the Instructor model and projected down to a 2D space via TSNE.

    Prompt: "Represent the hagiography for clustering by historical era, location, and theological contribution, ignoring common names"
